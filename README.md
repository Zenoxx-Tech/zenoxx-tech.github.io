Welcome to the official hub of **ByZenoxxx**! This is a personalized web portal designed for viewers to access my live stream, schedule, and setup in one place.

## 🇬🇧 English Description

### ✨ Features
* **Live Stream Integration:** Watch the Twitch stream directly with a toggleable chat.
* **Interactive Background:** A futuristic data-mesh background that reacts to mouse movements and clicks.
* **Dynamic Schedule:** View my weekly streaming hours.
* **Setup Showcase:** Explore the gear I use with an interactive grid.
* **Dark/Light Mode:** Full theme support for comfortable viewing.

### 🛠️ Built With
* HTML5 / CSS3 (Custom Properties)
* Vanilla JavaScript (Canvas API for the background)
